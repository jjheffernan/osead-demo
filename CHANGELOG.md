## [0.1.0] — 2026-07-15

OSEAD coastal real-estate template scaffold.

### Highlights

- Forked from Astro Cloudflare starter; React islands replaced with Svelte 5
- Dual inventory: weekly rentals + homes for sale
- Markdown `properties` / `markets` collections (OBX, Jersey Shore, Delaware Beaches)
- Routes: `/rentals`, `/sales`, `/markets`, `/properties/[slug]`
- SEO helpers: VacationRental + RealEstateListing JSON-LD
- AFK docs + after-hours / caveman / ponytail skills installed
